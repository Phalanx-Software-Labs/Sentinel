================================================================================
SENTINEL — PHASE 5: PACKAGING & PORTABILITY
================================================================================
Goal: Distribute Sentinel as a standalone installer and optionally port to
handheld/lightweight platforms (.sh script or similar for non-Windows).
================================================================================


--------------------------------------------------------------------------------
1. IN SCOPE
--------------------------------------------------------------------------------
- Standalone installer:
  - Package as .exe (e.g. PyInstaller, cx_Freeze) for Windows.
  - Single-file or minimal-folder install; no Python required on user machine.
  - Installer (e.g. NSIS, Inno Setup) for professional distribution.
- Optional handheld /.sh port:
  - Lighter workload: script-based quick check for Linux/Raspberry Pi or
    similar environments.
  - .sh script that runs Python if available, or minimal shell equivalent.
  - Focus: quick check only for handheld use; full sweep may be optional.


--------------------------------------------------------------------------------
2. OUT OF SCOPE FOR PHASE 5
--------------------------------------------------------------------------------
- Full feature parity on non-Windows (handheld port may be subset).
- Host integration on handheld (Mimic integration remains Windows-focused
  unless Mimic also ports).


--------------------------------------------------------------------------------
3. DELIVERABLES
--------------------------------------------------------------------------------
- Windows .exe build and installer.
- Optional: .sh quick-check script for Linux/handheld.
- Release packaging and versioning.
- User documentation (install, run, interpret results).


--------------------------------------------------------------------------------
4. KEY TECHNICAL NOTES
--------------------------------------------------------------------------------
- PyInstaller/cx_Freeze: bundle Python + deps; test on clean Windows VM.
- .sh port: may use different drive enumeration (e.g. /dev/sdX, mount points).


--------------------------------------------------------------------------------
5. ESTIMATE
--------------------------------------------------------------------------------
- Windows packaging: ~2 days (build, test, installer).
- Handheld port: ~1–2 days if scope is quick check only.


================================================================================
END OF DOCUMENT — PHASE 5
